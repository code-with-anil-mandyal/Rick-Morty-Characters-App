# 📱 Rick & Morty Characters App

A modern Android application that demonstrates **Clean Architecture**, **MVVM**, **Jetpack Compose**, **Paging 3**, **Coroutines**, and **Hilt Dependency Injection**.

The app fetches characters from the **Rick & Morty API** and displays them in a paginated list with a dedicated **character details screen**.

This project is built to showcase **modern Android development best practices and scalable architecture**.

---

# 🚀 Features

- 📃 Display **paginated list of characters**
- 👤 View **character details**
- ⚡ Efficient data loading using **Paging 3**
- 🔄 Reactive programming with **Kotlin Coroutines & Flow**
- 🧩 Clean Architecture with **MVVM**
- 💉 Dependency Injection using **Hilt**
- 🌐 REST API integration using **Retrofit**
- 🎨 Modern UI built with **Jetpack Compose**

---

# 🏗️ Architecture

This project follows **Clean Architecture principles** with clear separation of concerns.

### Architecture Flow

UI (Jetpack Compose)
↓
ViewModel
↓
UseCase
↓
Repository
↓
Remote API

This structure helps in:

- Better **code maintainability**
- **Separation of concerns**
- Easier **testing**
- Scalable architecture for large applications
